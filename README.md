# 📚 Estruturas de Dados

![Status](https://img.shields.io/badge/Status-Concluído-success)

## 💻 Sobre o Projeto
Repositório dedicado ao armazenamento de programas e algoritmos desenvolvidos durante a disciplina de Estruturas de Dados no IFBA. Contém implementações lógicas para organização e manipulação de informações.

## 🛠️ Tecnologias Utilizadas
* Java
* PHP

## 🚀 Como Executar
1. Clone o repositório.
2. Navegue até o diretório do arquivo desejado via terminal.
3. Compile e rode o código diretamente no terminal (ex: `javac Arquivo.java` e `java Arquivo` para os projetos em Java, ou `php arquivo.php` para os em PHP).
